# ビットコインマイニングプール - ドキュメント

このドキュメントはビットコインマイニングプールの実装に関する技術的な解説を提供します。

## 主要コンポーネント

1. **Pool** - マイニングプールのコア機能を提供するクラス
2. **StratumServer** - Stratumプロトコルを用いたマイナーとの通信を担当するクラス

## システム構成

このマイニングプールは以下のコンポーネントで構成されています：

1. マイナー接続と通信管理（StratumServer）
2. ブロックテンプレート管理と配布（Pool）
3. シェア検証とブロック提出（Pool）

## 動作フロー

1. `main.py`からプールとStratumサーバーが起動されます
2. マイナーがStratumサーバーに接続し認証します
3. プールが定期的にビットコインノードからブロックテンプレートを取得します
4. 新しいテンプレートが取得されるとマイナーに配布されます
5. マイナーはシェア（部分的な解答）を提出します
6. プールはシェアを検証し、有効なら記録します（ソロマイニングのため、報酬分配はありません）
7. ブロックが見つかった場合、ビットコインネットワークに提出されます

各コンポーネントの詳細な機能については、それぞれのドキュメントファイルを参照してください。
